## 📝 采购建议

1. **提前3天**：干货类（羊肚菌、香料包、药材、鱼子酱）和调味料可网购
2. **提前2天**：肉类（牛腱子、猪舌、猪肚需卤制浸泡过夜）、风吹肉、鸭胚（需腌制+风干24小时）
3. **提前1天**：海蜇头（需泡发4小时去咸）、花甲（需吐沙）、卤水拼盘食材（提前卤好）
4. **当天采购**：活海鲜（龙虾、膏蟹、鲍鱼——必须当天鲜活）、豌豆苗、莴笋等蔬菜
5. **全程冷链**：鱼子酱、酸奶油、黑松露需冰箱保存
